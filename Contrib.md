# Contributing to EL_Sem1

Thanks for your interest in contributing to this IoT Plant Monitoring project!

## Getting Started

1. **Fork this repository** to your own GitHub account.
2. **Clone the project** to your machine:
   ```bash
   git clone [https://github.com/ArchofSignals/EL_Sem1.git](https://github.com/ArchofSignals/EL_Sem1.git)
   How to Contribute
Create a new branch for your feature or fix:

Bash

git checkout -b feature/amazing-feature
Make your changes in the code.

Commit your changes with a clear message:

Bash

git commit -m "Added error handling to server.py"
Push to your branch:

Bash

git push origin feature/amazing-feature
Open a Pull Request in this repository.

Coding Style
Python: Please follow PEP 8 standards.

C++: Ensure variable names are descriptive (e.g., temperatureValue instead of t).
